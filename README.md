# URL Shortener
This app allows you to create short url links for easy managment.

## Installation
To use it just write this command
```bash
npm i
```

## Execution
To run de app you need 2 instances
1st:
```bash
npm run dev
```
2nd:
```bash
npm run server:dev
```
